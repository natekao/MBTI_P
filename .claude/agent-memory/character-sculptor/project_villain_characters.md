## 反派一：德叔（林德仁）— 人類反派

- 67歲，村莊護林祭師，白髮慈祥，左手小指有舊疤
- **核心動機**：非純粹邪惡，而是扛了五十年謊言後再也回不了頭的老人，以「保護村莊」之名延續小明祖先偽造虎鬼作亂的謊言
- 12歲時發現真相、選擇燒毀記錄，此後成為謊言守護者
- 曾與小花阿嬤有過正面對峙，阿嬤選擇將真相藏進木箱而未公開，德叔欠她一筆還不清的債
- 在小明父母失蹤後，德叔確保消息不外洩，對小明有歉疚但更有威脅感
- **對抗方式**：不使用暴力，而是以關心、長輩勸告、操弄輿論讓小明懷疑揭露真相的正當性
- **故事張力**：代表「人類選擇的沉默」，試圖讓小明「放下」

**Why:** 故事已確立小明祖先的罪惡，需要一個體現「制度性謊言延續」的人類反派，非單純邪惡，而是有複雜動機。

**How to apply:** 後續劇情中德叔應透過間接手段施壓，而非正面對抗；他可能掌握小明父母更多未揭露的秘密。

## 反派二：霧中影（餘靈）— 靈界反派

- 虎鬼被護身符消滅後，殘留怨氣與森林記憶交融而生的新存在
- 非虎鬼本身，是牠死亡的「回聲」，誕生自護身符爆發那一刻
- **外貌**：霧氣凝成的女性輪廓，銀白色眼睛，臉孔會折射出對方內心最深的牽掛；路過之處草木顏色暫時褪去
- **核心特質**：不帶憤怒只帶未竟真相，不說謊，以困惑與赤裸誠實直擊人心
- 承載虎鬼百年記憶，但沒有繼承憤怒（憤怒已在消滅時燃燒殆盡）
- 對小明有先天連結（小明是她感知到的第一個人），對小花家族有「認出同類」的感應，對岑岑有特殊的溝通可能
- **渴望**：讓虎鬼的故事被記得、被承認，這是牠存在的根本目的
- **若壯大而不得完成**：靈氣擴散影響自然環境，逼使村民面對歷史
- **故事張力**：代表「真相本身的呼喊」，逼迫小明「面對」

**Why:** 虎鬼被消滅後留下靈氣真空，需要一個新的超自然存在承載未解的道德問責，並給小明的自責一個可以對話的出口。

**How to apply:** 霧中影應以漸進方式現形，先透過夢境/畫作接觸小花、岑岑，最後才直面小明；德叔會試圖消除牠。

---

## 兩個反派的關係動態

- 德叔（人類沉默）vs 霧中影（真相呼喊）構成故事核心張力軸
- 小花的家族知識是連結兩者的鑰匙
- 小明夾在中間：德叔讓他「放下」，霧中影逼他「面對」
